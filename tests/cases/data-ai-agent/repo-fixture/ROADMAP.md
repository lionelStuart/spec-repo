# Roadmap

## Milestones

### M1

- Scope: dataset upload, registry bootstrap, schema profiling
- Exit Criteria: users can register files and inspect normalized schema

### M2

- Scope: natural-language analysis planner and execution trace
- Exit Criteria: users can ask analytical questions and receive traceable answers

### M3

- Scope: chart recommendation and chart generation
- Exit Criteria: users receive chart suggestions backed by execution plans
